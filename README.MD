# blog-ql

<p>Simple blog application. With multiple views - home, article category view and article view.</p>

<p>Technologies used:</p>
<ul>
    <li>PHP 8</li>
    <li>MySQL 8</li>
    <li>GraphQL</li>
    <li>Laravel v9.37</li>
    <li>Vue.js v3.2</li>
    <li>TailwindCSS v3.1</li>
    <li>Docker and Docker Compose</li>
</ul>
